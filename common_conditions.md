# Выполнение работы
Разрешается использование средств ИИ для разработки программного кода.
В этом случае студент должен провалидировать код, проверить его работу.
Использование ИИ не дает преимущества в знании того, что происходит в коде, поэтому студент должен изучить каждую строку кода и ее необходимость. 
Также ИИ при неправильной работе с кодом усложняет структуру программы и вносит непонятные изменения, которые будут заметны и отправлены на доработку.

Запрещается прямое копирование кода, т.к. многие вещи, которые придумывает ИИ, являются избыточными, и повторяемость этого в других отчетах дискредитирует выполняющего.

# Оформление отчета
Оформление по ГОСТу.

Обязательно указывать программный код из IDE в виде рисунков.
Также вставить программный код в приложение текстом.

Разбивать рисунки кода осмысленно с описанием того, что выполняется в данном участке кода.

Сам по себе код не является результатом работы, поэтому необходимо вставлять скриншоты логов программы, демонстрирующие результаты.
