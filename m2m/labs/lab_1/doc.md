# Задание

Варианты:
| Вариант | Сервис | Способ               | Утилита                      |
| ------- | ------ | -------------------- | ---------------------------- |
| 1       | Kafka  | Docker (compose)     | Kafdrop                      |
| 2       | MQTT   | Docker (compose)     | Доп. утилита от брокера MQTT |
| 3       | NATS   | Docker (compose)     | natscli                      |
| 4       | Kafka  | Системное приложение | Kafdrop                      |
| 5       | MQTT   | Системное приложение | Доп. утилита от брокера MQTT |
| 6       | NATS   | Системное приложение | natscli                      |

Необходимо развернуть сервис, настроить конфигурацию так, как указано на официальных сайтах сервисов.

В том случае, если в варианте способом указано системное приложение, то, как правило, разворачивание происходит стандартной установкой приложения (.exe в Windows, apt/bin в Linux).

Если указан Docker, то необходимо подготовить docker_compose.yaml c сервисом и развернуть в Docker-контейнере из команды docker compose.

Также необходимо "в комплекте" развернуть системные утилиты для просмотра, управления системой.

Внимание:
* развертывание в серед Windows может быть неудачным
* linux можно использовать в Виртуальной машине / WSL / как основная ОС
* после развертывания, проверить успешность путем реализации клиента с отправкой сообщения (клиент на python скрипте), клиента-слушателя реализовать через cli-инструменты; успешностью будет оптравка сообщения через python и получение в терминале
